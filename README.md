# CustomDriveITT

## Objetivo

Desarrollar un sistema web interactivo de configuración de autos, que permita a los usuarios personalizar un vehículo desde cero (marca, modelo, color, rines y extras), calcular el costo total dinámicamente y visualizar una vista previa del carro en tiempo real, utilizando buenas prácticas de programación y el patrón de diseño Decorator para mantener el código escalable y modular.
